# Roadmap: AI Brainstorming PRD Copilot

## Overview

本路线图围绕“把模糊产品想法持续压实成可确认、可执行的 PRD”展开，按能力依赖与产品优先级拆成五个阶段：先把引导节奏和选项式交互做对，再补足矛盾/缺口诊断深度，然后把澄清结果沉淀成带证据的首稿，接着完成按章节增量收敛的 PRD 编排，最后补质量复核、导出与回放能力。
